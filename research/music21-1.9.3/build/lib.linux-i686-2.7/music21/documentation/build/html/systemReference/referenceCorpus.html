List of Works Found in the music21 Corpus¶

The following list shows all files available in the music21 corpus and available through the virtual corpus. To load a work from the corpus, provide the file path stub provided.

For example:

>>> from music21 import corpus
>>> s = corpus.parse('bach/bwv108.6.xml')

Aird’s Airs¶

To get all works by Aird’s Airs, the getComposer() function can be used to get all file paths.

For example:

>>> from music21 import corpus
>>> paths = corpus.getComposer('airdsAirs')
